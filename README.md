### user ro'yxatgan o'tkazish

| endpoint     | izoh                                                                             | method |
| ------------ | -------------------------------------------------------------------------------- | ------ |
| `/users/`    | barcha userlar ro'yxati                                                          | GET    |
| `/users`     | yangi user yaratadi (name, phone, username, password, user_image)                | POST   |
| `/users/:id` | mavjud userni `id`'si bo'yicha izlab topadi va berilgan ma'lumtlarini yangilaydi | PUT    |
